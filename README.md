Landing Page profesional para servicios de **Limpieza de Bases de Datos y Automatización (Procesos ETL)**. Diseñada pensando en la conversión, aplicando principios de Fluidez Cognitiva, el Efecto Halo y reglas de UX orientadas a PyMEs tradicionales y negocios B2B.

## 🎯 Objetivo
Ofrecer una interfaz ultra limpia, rápida y directa que comunique "paz mental" y ahorro de tiempo operativo frente al caos del trabajo manual con planillas dispersas.

## ✨ Características Principales
- **Diseño "Premium" Minimalista:** Colores calculados para reducir la fatiga visual (Gris Pizarra, Gris Carbón) y resaltar CTAs (Verde Esmeralda).
- **Escalabilidad Visual:** Tipografía fluida (`rem`) que escala proporcionalmente en monitores `1440p` y `4K` (1920p+) usando Tailwind CSS V4.
- **Microinteracciones Sutiles:** Hover effects en botones, tarjetas (cards) informativas con efecto de elevación suave e íconos reactivos (`lucide-react`).
- **Formulario Funcional Integrado:** Envíos reales configurados directo a correo mediante la API de **Web3Forms** (Sin necesidad de Backend).
- **Responsive Design 100%:** Experiencia impecable desde dispositivos móviles hasta pantallas interactivas de gran formato.

## 🛠️ Stack Tecnológico
- **Frontend Framework:** React 19
- **Build Tool:** Vite
- **Estilos:** Tailwind CSS v4.0
- **Iconografía:** Lucide React

## 🚀 Despliegue Local (Running Locally)

Sigue estos pasos para correr el proyecto localmente:

1. Clonar el repositorio:
```bash
git clone https://github.com/capano-andres/capanoandres-web.git
```

2. Ir al directorio del proyecto:
```bash
cd capanoandres-web
```

3. Instalar las dependencias:
```bash
npm install
```

4. Configurar Web3Forms (Para el envío de emails de la sección Contacto):
- Duplica el archivo `.env.example` y renómbralo a `.env`.
- Abre `.env` y coloca tu llave generada en Web3Forms:
  ```env
  VITE_WEB3FORMS_ACCESS_KEY=tu_api_key_aqui
  ```

5. Levantar el entorno de desarrollo:
```bash
npm run dev
```

6. Empaquetar para producción (Build):
```bash
npm run build
```
